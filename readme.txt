Show the best places to stay near your event

== Description ==

Show your guests the best places to stay right near your event. Just set your event address to show all the accommodation nearby. Guests can search on the map by preference and then click straight through to booking. Easy and convenient!


*   Easy to set up and simple to use
*   Huge variety of accommodation types to suit all needs
*   Optimized for both mobile and desktop
*   Customizable design
*   Available in multiple languages

== Installation ==

1. Choose to add a new plugin, then click upload
2. Upload the hotels-and-airbnbs-map zip
3. Activate the plugin through the 'Plugins' menu in WordPress
4. To activate the widget drag it to a sidebar or click on it
